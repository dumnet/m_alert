# M Alert
M Alert - A very lightweight modal (No additional javascript library or css required)

<b>Basic Usage</b>

\<script src="m_alert.js">\</script>

\<!-- Create Modal --> <br>
\<script><br>
        m_alert("This is a test message"); <br>
\</script>
